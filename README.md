# Flappy Bird — ColorStack Workshop

I built and taught this workshop in April 2026 for ~10 college students at ColorStack, walking them through core Python concepts by building a complete game from scratch with Pygame.

A beginner-friendly Python game built with Pygame. In this workshop you'll build Flappy Bird from scratch, learning core programming concepts along the way.

---

## What You'll Learn

- Variables, functions, and conditionals
- Game loops and event handling
- Collision detection
- Working with a real Python library (Pygame)

---

## Prerequisites

- Python 3 installed on your machine
- A code editor (VS Code recommended)
- No prior game dev experience needed!

---

## Setup

**1. Make sure Python is installed**
```bash
python --version
```

**2. Create and activate a virtual environment**
```bash
python -m venv venv

# Mac/Linux
source venv/bin/activate

# Windows
venv\Scripts\activate
```

**3. Install Pygame**
```bash
pip install pygame
```

**4. Run the finished game to see what you're building**
```bash
python main.py
```

---

## Workshop Files

| File | Description |
|---|---|
| `main.py` | The completed game — peek here if you get stuck |
| `main_template.py` | Your starting point — fill in the TODOs! |
| `assets/` | Images, sounds, and font used by the game |

---

## How to Play

- Press **Space** to make the bird flap
- Avoid the pipes and the floor
- Try to beat your high score!

---

## Tips for Beginners

- Read the `# TODO` comments in `main_template.py` — they tell you exactly what each function needs to do
- Run your code often to catch errors early
- If something breaks, check the error message — it usually tells you the line number
- Ask questions! That's what the workshop is for
